汇聊接口文档

    聊天记录分页查询接口-CRM

    接口类型 JSON

    请求地址 /crm/msg/get-page

    请求方式 POST

    请求参数

    参数类型必选说明
    uid1array是消息一方uid，最多不能超过1000个
    uid1_roleint否消息一方角色，枚举值（1：发送方，2：接收方）
    uid2array否消息另一方uid，最多不能超过1000个
    typeint否消息类型，枚举值（1：诊股，11：文本，12：语音，13：图片，14：视频，15：文件）
    start_msgtimeint否消息起始时间戳
    end_msgtimeint否消息结束时间戳
    keywordsstring否关键词，仅对文本消息有效
    stateint否质检状态，枚举值（1：未质检，2：已质检）
    updaterint否质检人id
    pageint否页码，默认值：1
    page_sizeint否每页条数，默认值：50

    响应格式 JSON

    响应内容

    参数类型说明
    codeint响应码，枚举值（0：成功，>0：失败）
    messagestring响应描述
    dataobject响应数据，详见data说明

    data说明

    参数类型说明
    listarray数据列表，按msgid降序排列，详见list说明
    total_sizeint总条数
    total_pageint总页数

    list说明

    参数类型说明
    msgidint消息id
    typeint消息类型
    msgobject消息体，详见 proto
    msgtimeint消息时间戳
    senderint发送人uid
    receiverint接收人uid
    stateint质检状态
    updaterint质检人id
    update_timeint质检时间戳
