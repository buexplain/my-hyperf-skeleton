汇聊接口文档

    聊天记录设置已读接口-CRM

    接口类型 JSON

    请求地址 /crm/msg/update

    请求方式 POST

    请求参数

    参数类型必选说明
    msgidarray是消息id，最多不能超过1000个
    updaterint是质检人id

    响应格式 JSON

    响应内容

    参数类型说明
    codeint响应码，枚举值（0：成功，>0：失败）
    messagestring响应描述
    dataobject响应数据，详见data说明

    响应示例

    {
    "code": 0,
    "message": "成功",
    "data": {}
}
