汇聊接口文档

    单人/双人聊天记录查询接口

    接口类型 JSON

    请求地址 /msg/get-slice

    请求方式 POST

    请求参数

    参数类型必选说明
    uid1int是消息一方uid
    uid2int否消息另一方uid
    start_msgtimeint否消息起始时间戳
    end_msgtimeint否消息结束时间戳
    keywordsstring否关键词，仅对文本消息有效
    offset_msgidint否起始消息id
    sizeint否返回条数，默认值：50

    响应格式 JSON

    响应内容

    参数类型说明
    codeint响应码，枚举值（0：成功，>0：失败）
    messagestring响应描述
    dataobject响应数据，详见data说明

    data说明

    参数类型说明
    listarray数据列表，从offset_msgid开始按msgid降序排列（返回数据包含offset_msgid，需要注意去重），详见list说明
    total_sizeint总条数

    list说明

    参数类型说明
    msgidint消息id
    typeint消息类型
    msgobject消息体，详见 proto
    msgtimeint消息时间戳
    senderint发送人uid
    receiverint接收人uid
